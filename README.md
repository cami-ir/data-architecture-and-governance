# data-architecture-and-governance

This repository contains the code for the Philanthropy data architecture and governance pipeline.

**Documentation for this project lives in the Institutional Research Obsidian vault**, organized under the Data Governance Hub. This includes:

- Architecture strategy and diagrams
- Data owner & steward guidelines
- Data steward engagement plan
- Alumni contact reconciliation pilot
- Governance discrepancy pipeline architecture
- Discrepancy resolution workflow
- Arryval → Veracross data transfer plan

## What's in This Repo

- `governance-discrepancy-pipeline/sample-code/` — Cloud Function and pipeline scripts for the discrepancy detection and Veracross write-back
